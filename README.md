# Item Response Theory Demo Collection

The package can be installed and loaded as:
```{r}
install.packages("irtDemo")
library(irtDemo)
```
Example usage:
```{r}
irtDemo("eapmap")
irtDemo("mirt")
irtDemo("mle")
irtDemo("est[]2pl")
irtDemo("est[]3pl")
```
